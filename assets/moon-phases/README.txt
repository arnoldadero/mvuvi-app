Moon Phase Images

This directory contains placeholder images for the different moon phases. In a production environment, you should replace these with high-quality images that clearly represent each moon phase.

Required moon phase images:
- new-moon.png
- waxing-crescent.png
- first-quarter.png
- waxing-gibbous.png
- full-moon.png
- waning-gibbous.png
- last-quarter.png
- waning-crescent.png

For best results, use PNG images with transparency, sized at 256x256 pixels.
